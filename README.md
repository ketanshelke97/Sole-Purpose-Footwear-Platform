# Sole Purpose - Unified Footwear Platform üëü

**Sole Purpose** is a specialized e-commerce platform designed to bridge the gap between style, comfort, and sustainability. It solves the common problem of online sizing inconsistencies and lack of guidance for specific foot conditions (e.g., Flat Feet, Bunions).

## üöÄ Key Features

* **Smart Size Converter:** A JavaScript-based algorithm that converts CM measurements to standard US, UK, and EU sizes instantly.
* **Orthopedic Health Center:** Dedicated product filtering for specific foot conditions (Plantar Fasciitis, Wide Feet, etc.).
* **Local Brand Spotlight:** A curated marketplace promoting indigenous Indian footwear brands (e.g., Paragon, Liberty).
* **Sustainability Initiative:** An integrated locator for shoe recycling centers to promote a circular economy.
